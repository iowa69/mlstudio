Welcome

  1. Click Browse and pick a folder of .fasta assemblies.
  2. Pick an MLST or cgMLST scheme — or build one from a reference (schemes build-adhoc).
  3. Click Analyze.
  4. Upload a metadata CSV to color the tree. Adjust the cluster halo distance to highlight outbreak clusters.

Re-running on the same folder is incremental — only new isolates get typed.
